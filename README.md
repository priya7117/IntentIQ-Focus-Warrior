
# 🚀 IntentIQ (Focus Warrior)

IntentIQ (Focus Warrior) is an Android productivity and digital wellness application designed to help users understand and improve their digital habits.

The application tracks app usage, categorizes activities into meaningful productivity groups, and provides insights that help users stay focused and maintain consistency through streaks and gamification.

## ✨ Features

* 📱 Real-time app usage tracking
* 📊 Productivity analytics dashboard
* 📚 Activity categorization (Learning, Work, Entertainment)
* 🔥 Focus streak tracking
* 🎮 Gamified productivity experience with Lumis
* 📈 Daily progress monitoring
* 🌙 Modern dark-themed UI
* ⚡ Background activity monitoring

## 🛠️ Tech Stack

* Kotlin
* Jetpack Compose
* Android Accessibility Services
* Android Background Services
* Broadcast Receivers
* SharedPreferences
* Material Design 3

## 🎯 Objective

The primary goal of IntentIQ is to help users:

* Understand where their screen time is spent
* Improve productivity and focus
* Build consistent habits through streaks
* Reduce unnecessary distractions
* Develop healthier digital behavior
